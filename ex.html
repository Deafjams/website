My interests

Math is pretty neat. I write code to solve math problem; that is, solve partial differential equations numerically. There are many complicated problem in which analytic solutions cannot be found, approximate solutions to these problems can be established with numerical methods. So the big question is why do we care? Well, for many problems, approximations are good enough. We in the numerical analysis community always value getting to the best answer in the quickest amount of time. We are alright with the "good" enough answer though.

I'm also very interested in computer science. I like writing bots. See my twitter account, it is fully automated. I mean, I actually update my twitter close to 500 times a day. I have no life (Sarcasm). I think the concept of getting things done in parallel is a very neat idea. So, I like running a lot of scripts while I do math. Life is all about optimizing time.
